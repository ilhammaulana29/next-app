# Next.js Project

Dokumen ini berisi panduan cepat untuk menjalankan proyek setelah mendownload kode dari GitHub.

---

## Persyaratan Versi

- **Node.js**: versi **18.x atau 20.x** direkomendasikan.  
  Next.js 14 dan React 18 bekerja optimal di versi ini.
- **npm / yarn / pnpm**: versi terbaru disarankan.
- Proyek ini menggunakan **TypeScript**, **TailwindCSS**, dan **Ant Design**.
- Pastikan backend (BE) yang terhubung juga berjalan sesuai instruksi, agar frontend dapat melakukan API request.

---


## 1. Clone Repository

Pertama, clone repository ke komputer lokal:

```bash
git clone https://github.com/ilhammaulana29/next-app.git
cd next-app
```


## 2. Install Dependensi bisa pakai npm, yarn dan pnpm

```bash
# menggunakan npm
npm install

# menggunakan yarn
yarn install

# menggunakan pnpm
pnpm install
```


## 3. Jalankan Projek

```bash
# Menggunakan yarn
yarn dev

# Atau menggunakan npm
npm run dev
```
Server akan berjalan di `http://localhost:3000`

## 4. Pastikan BE sudah dijalankan

```bash
# Download BE di link berikut:
https://drive.google.com/file/d/1mJAS-AFxj3NU9jvPD9Ue9Z56ssX4Mcrs/view?usp=sharing

# Ikuti langkah-langkah menjalankan BE seperti yang ada di file README.md di BE
```
